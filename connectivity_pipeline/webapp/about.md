# Connectivity Pipeline

The **Connectivity Pipeline** is a geospatial analysis tool designed to measure urban accessibility and business connectivity at the neighbourhood level. It operationalises two composite indices: the **People Connectivity Index (PCI)** and the **Business Connectivity Index (BCI)**, using a Hansen gravity model applied to a hexagonal H3 grid. PCI captures how well residents of each hexagon can reach key urban amenities (health, education, parks, transit, food, community facilities) across a multi-modal transport network of walking, cycling, driving, and public transit, weighted by local income to reflect the true cost burden of travel. BCI measures how well businesses can access the three key inputs of economic activity: consumer markets, labour pools, and supply-chain services, across component-specific networks, producing a holistic picture of commercial accessibility that complements the resident-centred view.

The tool is intended for urban planners, researchers, and policy analysts who need to identify spatial inequities in access to amenities and economic opportunity. Users can interactively adjust model parameters (decay coefficients, amenity weights, BCI component weights) and immediately see how results change without rebuilding the underlying network. The **Diagnostics** tab provides network validation and isochrone visualisations to verify model inputs; the **Sensitivity** tab runs one-at-a-time (OAT) analysis to quantify how robust the index scores are to parameter choices. Results are automatically saved to disk and can be restored across sessions, so long-running computations only need to be performed once per city.
